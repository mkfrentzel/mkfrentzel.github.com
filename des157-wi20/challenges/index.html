Why we use the Normalize CSS Script

We use the Normalize css script because it helps create a standardized theme across all web browsers, instead of simply wiping the slate clean completely and destroying your design if you view it in a different browser. It is better than other stylesheets because you can target specifically which sections of the code that you want to stylize, by adding only certain parts of the code or you can add the entire snippet and use it's styling across all elements, editing as you please.

What is Semantic Markup

Semantic Markup is when the html elements add actual meaning to the code. For example, the paragraph tag indicates that a paragraph will be starting. This means that the semantics of the code align with the presentation of the code in a format that we understand based on the meaning of the word.
