Exploring AOS Scroll Library

For this project I wanted to explore the AOS scroll effect library because I plan to use a lot of scrolling in my final project. Scroll down to see some of my favorite effects.

Fade Up Example. This particular fade happens when the div is in the top-center of the page.

Fade Right Example. This example has an ease added to help transition the fade.

Zoom Out Down Example. This example zooms out and down, and I added the placement so it occurs when the div is at the top-center of the page.

Flip Down Example. This flips onto the page when the div is at the bottom of the page.
